# 🏖️ Fortaleza Insider | Landing Page de Turismo

Bem-vindo ao **Fortaleza Insider**! Este projeto é uma landing page moderna e imersiva desenvolvida para apresentar pontos turísticos icônicos de Fortaleza/CE, utilizando técnicas avançadas de Web Design de 2026.

## 🚀 O Projeto
O objetivo foi criar uma experiência de usuário (UX) de alto nível, simulando guias de turismo de luxo. O site conta com:
- **Background em Vídeo:** Drone footage da Beira Mar para imersão imediata.
- **Menu Dinâmico:** Header com efeito *glassmorphism* que alterna opacidade via JavaScript ao rolar a página.
- **Curadoria de Conteúdo:** Cards interativos com efeitos de *hover* (zoom) apresentando a Ponte Metálica, o Planetário e a Catedral.
- **Design Responsivo:** Adaptado para leitura fluida em desktops e dispositivos móveis.

## 🛠️ Tecnologias Utilizadas
- **HTML5:** Estrutura semântica para SEO.
- **CSS3:** Estilização com variáveis (root), Flexbox e Grid Layout.
- **JavaScript (Vanilla):** Lógica para manipulação do DOM e eventos de scroll.
- **Google Fonts:** Tipografia selecionada (Montserrat e Nunito Sans).
- **FontAwesome:** Ícones sociais e de navegação.

## 📖 O Processo de Desenvolvimento
O desenvolvimento seguiu as melhores práticas de Front-End:
1. **Concepção Visual:** Foco em cores que remetem ao litoral e fontes modernas.
2. **Otimização de Mídia:** Uso de arquivos locais (`.mp4` e `.jpg`) para garantir performance e carregamento instantâneo.
3. **Deploy Contínuo:** Integração do repositório GitHub com o **Netlify** para hospedagem automática.
